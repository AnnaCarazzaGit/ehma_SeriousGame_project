# Requirements for ehma_SeriousGame_project
# This is NOT a Python project: since this is a Unity project, "requirements"
# here means the editor, version, and packages needed to open and build it.

## Unity Editor
Unity 6000.2.6f2 (Unity 6)
- Install via Unity Hub: https://unity.com/download
- If this exact version is no longer available, use the latest patch of the
  same release (6000.2.x) or a later Unity 6 LTS version.

## Recommended Editor modules (from Unity Hub during installation)
- Build support module for your target platform (e.g. Windows Build Support / Mac Build Support)

## Installation
1. Install Unity Hub.
2. Through Unity Hub, install Unity editor version 6000.2.6f2.
3. In Unity Hub: Add → select the `unityProject/` folder.
4. Open the project: Unity automatically resolves all the packages listed
   above by reading `Packages/manifest.json` — no manual installation needed.

## Third-party assets included in the project
- "Ninja Adventure - Asset Pack" (character sprites, enemies, environments)
- TextMesh Pro (bundled/managed by Unity itself via TMP Essential Resources,
  prompted on first import if not already present)

Check the usage licenses for these assets before publishing the project.
